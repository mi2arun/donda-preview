Purchase Invoice (Import)

Created when diamonds are received & verified. On confirmation it auto-creates Rough Lot Master records and accounting entries.

Build model: one purchase invoice = one won lot (parent 1-26); the lines below are its sub-lots 1-26-A/B/C, each a stock unit. See Purchase Entry — Design for the build-ready detail.

Expenses

Expenses DYNAMIC

Expense Type (Freight, Import Duty, Insurance, Brokerage, Other), Amount, Remarks. Add any number of rows.

Expense allocation: expenses are allocated proportionally across line items by value. Each Rough Lot's inventory value = Line Amount + Proportional Expense Share.
