Future Scope

Consignment / Memo trading. Not required now — Hong Kong buys rough outright only. Mumbai / Surat occasionally issue a rough memo for local rough sale. Later: memo issue/return, ownership-retained-until-sold tracking, memo-to-invoice conversion.
India local sales — GST per HSN code. For the India entity, local rough sales must compute GST by HSN code. Requires HSN master with GST rates, tax breakup (CGST/SGST/IGST), and GST-compliant invoice output.
Other planned modules: Manufacturing/Polishing, full Accounting ledgers, and live courier API integration. (Bidding is now in scope as the ERP-backed data store for the mobile app.)
